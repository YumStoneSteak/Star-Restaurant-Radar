# Star Restaurant Radar v1.0.0

Initial NSIS installer release of 스타 레스토랑 레이더.

## 포함 사항

- `@byeolsikdang` 최신 Instagram 게시물 이미지 확인 및 Windows 알림
- 설정한 시간 기준 Windows 작업 스케줄러 등록
- Instagram 로그인 세션 생성용 Chrome 창 열기
- GitHub Releases 기반 수동 업데이트 확인
- per-user NSIS 설치 및 silent update 지원
