# StarRestaurantRadar v2.0.0

StarRestaurantRadar v2.0.0은 첫 실행부터 메뉴 확인, 알림 표시, 업데이트 설치와 재실행까지의 흐름을 자동화하고, 설정 화면과 트레이 문구를 한국어 중심으로 정리한 주요 릴리즈입니다.

## 주요 변경 사항

- 자동 알림은 설정 시각부터 당일 1시간 이내에만 시도합니다.
- 다음 날 실행될 때 전날 게시물이나 날짜를 확인할 수 없는 이전 캐시를 알리지 않습니다.
- 설정 변경 시 Windows 작업 스케줄러 등록을 UI 스레드 밖에서 처리해 창 멈춤을 방지합니다.
- 업데이트 확인과 설치 파일 다운로드를 UI 스레드 밖에서 처리해 창 멈춤을 방지합니다.
- GitHub 최신 릴리즈의 설치 파일과 SHA-256 검증 파일을 자동으로 받아 검증한 뒤 무인 설치하고, 설치 완료 후 앱을 자동 재실행합니다.
- SHA-256 검증 파일이 없거나 해시가 일치하지 않으면 자동 설치를 중단합니다.
- 첫 실행 시 최신 메뉴를 자동으로 확인하고, 별도의 확인 완료 토스트 없이 메뉴를 찾으면 바로 본 알림을 표시합니다.
- 트레이 실행 안내와 주요 조회 오류 문구에서 남아 있던 영어 표현을 한국어로 정리했습니다.
- 창을 닫을 때 변경 사항이 없으면 작업 스케줄러를 불필요하게 다시 등록하지 않습니다.
- 작업 스케줄러 명령에 10초 제한을 두어 Windows 명령이 응답하지 않을 때 무기한 대기하지 않습니다.
- 종료 직전에 미적용 설정이 있으면 UI를 계속 응답 가능한 상태로 유지하면서 적용을 마친 뒤 종료합니다.
- 기본 알림 시각과 새 예약 작업의 기본값을 평일 오전 9시 30분으로 변경했습니다.
- 최근 확인 시각을 ISO 시간대 문자열 대신 `오늘 오전 9:17`, `어제 오후 4:05`처럼 읽기 쉬운 형식으로 표시합니다.
- 별 아이콘을 다중 해상도 ICO와 투명 PNG로 생성해 앱·트레이·토스트·EXE·설치 및 제거 프로그램에 통일했습니다.
- 설정창과 상세창을 흰색·선샤인 옐로·딥 네이비 기반의 작고 밝은 공통 테마로 개편했습니다.
- 자동 실행 체크 영역은 옅은 파랑 배경으로, 종료 버튼은 옅은 빨강 배경으로 분리해 카드 안에서도 명확히 구분되도록 했습니다.
- 바로 실행 버튼 네 개를 2×2 배열로 배치해 섹션 높이를 줄였습니다.
- 알림 시간 입력의 오작동하던 위·아래 화살표를 제거하고 키보드로 시간을 직접 입력하도록 단순화했습니다.
- NSIS 설치 마법사에 같은 별 아이콘과 밝은 헤더 이미지를 적용했습니다.

## 설치 파일

- `StarRestaurantRadar-Setup-v2.0.0.exe`
- `StarRestaurantRadar-Setup-v2.0.0.exe.sha256`

`dist\StarRestaurantRadar\StarRestaurantRadar.exe`는 설치 파일이 아니라 설치 패키지에 포함되는 앱 실행 본체입니다. 일반 배포와 설치에는 위의 버전이 포함된 `Setup` 파일을 사용합니다.

## 검증 항목

- Python 구문 검사 및 전체 단위 테스트
- 설정 적용 중 UI 응답성 스모크 테스트
- 트레이 종료 및 UI 레이아웃 스모크 테스트
- 100%·125%·150% 배율 UI 레이아웃 스모크 테스트
- 바로 실행 2×2 배치, 시간 입력 화살표 제거, 자동 실행·종료 대비 스타일 검사
- ICO 다중 해상도, PNG 투명도, 기존 아이콘 참조 제거 검사
- NSIS 설치 파일 생성과 SHA-256 검증
- 실제 GitHub 릴리즈 자산 다운로드와 SHA-256 검증
- 가상 최신 버전으로 다운로드·무인 설치·자동 재실행 전체 흐름 검증
- 별도 설치 경로에서 설정·상태 파일 보존 및 설치본 실행 확인
